# @openjiuwen/relay-core

Headless runtime contracts for OfficeClaw provider plugins.

This package exports:

1. `OfficeClawProviderPlugin`
2. `ProviderPluginRegistry`
3. `AgentService` and related runtime types

If you are building an external client provider package, start with:

1. [Build a Provider Plugin](../../docs/guides/build-provider-plugin.md)
2. [packages/core/src/plugin/types.ts](./src/plugin/types.ts)
3. [packages/core/src/plugin/registry.ts](./src/plugin/registry.ts)
